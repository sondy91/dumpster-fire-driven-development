Feedback System · Prototype Demo
Feedback System
Two types. One endpoint.
The floating widget captures structured feedback during demos. Inline reactions attach to specific cards and components — thumbs up/down with component context automatically included. Both POST to /feedback with kind: "widget" | "reaction".
Integration
Drop into any page with one line. Inline reactions are auto-discovered from data-feedback-component attributes — no extra JS needed.
// 1. Optional config before the script tag window.METRIX_FEEDBACK_CONFIG = { endpoint: 'http://localhost:8765/feedback', page: 'dora-dashboard', // auto-detected from URL if omitted userName: 'alice', // optional pre-fill }; // 2. Add the script <script src="feedback-widget.js"></script> // 3. Mark any element for inline reactions <div data-feedback-component="ai-recommendations" data-feedback-label="Was this useful?"> ...your card content... </div> // Widget auto-injects 👍 👎 buttons. Or call manually: MetrixFeedback.react('my-component', 'up');
