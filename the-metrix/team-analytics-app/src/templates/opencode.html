WPM & Time Assumptions

These values control how your CLI calculates manual time equivalents. Tune them to match your team's actual pace for accurate savings estimates.

Manual time formula:
Bash: (chars / (wpm × 5 / 60)) + exec_time per command
Read: read_time per file
Edit: edit_time per file
Search: search_time per search

These are the fallback values used when your CLI stdout doesn't include explicit time_saved figures.
