# The Metrix

A suite of tools for tracking developer productivity and team analytics.

## Components

- **daily-metrics** - CLI for tracking daily work (commits, PRs, Jira issues)
- **metrics-api** - REST API for storing and querying metrics
- **team-analytics-app** - Web dashboard for sprint metrics and velocity
- **splash-screen-tester** - UI components for feedback widgets

## Quick Start

1. Copy `.env.example` to `.env` in each component
2. Configure your Jira and GitHub credentials
3. Build and run the components

See README.md for detailed setup instructions.
